10.(2014·四川卷,2,3分,难度★)下列词语中,没有错别字的一项是(
　D　)
A.打蜡　　顷刻　　生死攸关　　口干舌噪
B.飙升　　印证　　贻养天年　　扶摇直上
C.巨擘　　清彻　　历久弥新　　所向披靡
D.皱褶　　荧屏　　风生水起　　精简机构
解析A项,“口干舌噪”应为“口干舌燥”;B项,“贻养天年”应为“颐养天
年”;C项,“清彻”应为“清澈”。

11.(2013·四川卷,2,3分,难度★)下列词语中,没有错别字的一项是(
　A　)
A.平添　　算账　　声名鹊起　　厉行节约
B.砥砺　　麦杆　　徇私舞弊　　好高骛远
C.硫璃　　称颂　　关怀备至　　有恃无恐
D.飞碟　　疏浚　　出尔反尔　　突如奇来
解析B项,“麦杆”应为“麦秆”;C项,“硫璃”应为“琉璃”;D项,“突如奇来
”应为“突如其来”。
12.(2013·湖北卷,2,3分,难度★)下列各组词语中,没有错别字的一组
是(　C　)
A.彷徨　　愁怨　　寂寥静默　　凄婉惆伥
B.顾盼　　精捍　　步履稳健　　风神潇洒
C.睿智　　禀赋　　崇高品质　　趋善避恶
D.辩难　　商榷　　典藉满架　　旁稽博采
解析A项,“凄婉惆伥”应为“凄婉惆怅”;B项,“精捍”应为“精悍”;D项,“
典藉满架”应为“典籍满架”。

13.(2013·江西卷,2,3分,难度★)下列词语中,没有错别字的一组是(
　D　)
A.松驰　　回溯　　卫戍　　皇天后土
B.辨认　　影牒　　荣膺　　残羹冷炙
C.豆蔻　　聘礼　　修葺　　金壁辉煌
D.城阙　　编纂　　恻隐　　亭亭玉立
解析A项,“松驰”应为“松弛”;B项,“影牒”应为“影碟”;C项,“金壁辉煌
”应为“金碧辉煌”。
14.(2013·湖南卷,2,3分,难度★)下列词语中,没有错别字的一组是(
　D　)
A.扮靓　　商贾　　关怀倍至　　余音绕梁
B.辐员　　魁梧　　天花乱坠　　彪炳千古
C.联袂　　眈误　　沧海一粟　　插科打诨
D.寒暄　　遴选　　克勤克俭　　针锋相对
解析A项,“关怀倍至”应为“关怀备至”;B项,“辐员”应为“幅员”;C项,“
眈误”应为“耽误”。
