⑤E　前后矛盾。“果不其然”指果然(强调不出所料)。句子前后表意相反，使用不当。
⑥√　“光怪陆离”形容现象奇特，色彩繁杂。
4．请判断下列成语使用的正误，并根据上面要求作答。
①沿着岷江且行且看，既能感受都江堰鬼斧神工、动人心魄的伟大，又能领略沿江两岸鲜为人知的民族文化，体会别样的风土人情。(　　)
②木船在风浪中剧烈地摇晃着，那人却稳稳地站立着，就像一个身怀绝技的骑士，骑在一匹桀骜不驯的野马上，任凭野马狂奔，他却泰然自若。(　　)
③他一直吞吞吐吐，欲言又止。大家诚挚的劝慰，使他消除了顾虑，他这才说出了难言之隐的苦衷。(　　)
④曾国藩的日记与家书，写这些个鸡栏、菜圃小事，与其说是给家人子弟看，不如说是给慈禧太后看的，以期在无形中消除她的疑心，表示自己不过是一个求田问舍的乡巴佬，以保全脑袋而已。(　　)
⑤在中原地区，深秋的天气忽冷忽热，最近几天因感冒到医院就诊的人已经人满为患了。(　　)
⑥海滨公园是附近居民喜爱的运动场所，在花海中无论是散步、慢跑还是骑车锻炼都令人神气十足。(　　)
答案：①√　“鬼斧神工”像是鬼神制作出来的。形容技艺高超精巧，不是人力所能达到的。
②√　“桀骜不驯”指性格凶暴倔强或态度傲慢不服从。在句中修饰“野马”是可以的。
③A　重复赘余。“难言之隐”难于说出口的藏在内心深处的事情。与“苦衷”语意重复。
④√　“求田问舍”多方购买田地，到处问询屋价；指只知道置产业，谋求个人私利；比喻没有远大的志向。
⑤A　重复赘余。“人满为患”指人太多，超过了可以容纳的限度，成了灾难。与前面的“人”重复。
⑥E　前后矛盾。“神气十足”形容摆出一副自以为高人一等而了不起的样子，带有傲慢的神气，而语境是“神定气闲”。
5．请判断下列成语使用的正误，并根据上面要求作答。
①七千余名莘莘学子在讨论中达成了共识：新时代的大学生应该具有勤于学习、善于思考、勇于实践、乐于奉献的精神。(　　)
②美国师出无名的伊拉克战争，造成10万平民死于非命，也让1700多名美国大兵丧生异域。(　　)
③著名作家冰心老人生前三令五申，青少年学生应该努力学好语文，有责任有义务维护祖国语言文字的纯洁性。(　　)
④日常交往中，平等是人与人之间投桃报李、礼尚往来的前提，高高在上、盛气凌人只会使人与人彼此疏离，产生隔阂。(　　)
