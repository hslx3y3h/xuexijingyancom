2020年学业水平考试语文对点备考学案：第四专题(2)

3．依次填入下列各句横线处的词语，最恰当的一项是(　　)
(1)调研组还实地________了武夷山市太子民俗文化企业，闽台商贸交流，以及闽台佛教文化交流的具体情况。
(2)我们很有信心在奥运马术比赛期间，为赛事提供________的天气预报。
(3)这篇论文写得精准透彻，主要得力于作者________了大量资料。
A．考察　精细　收集
B．考察  精细  搜集
C．考查  精密  搜集
D．考查  精密  收集
答案　B
解析　“考察”指实地观察调查或细致深刻地观察，“考查”指用一定的标准来检查衡量(行为、活动)。“精细”指精密细致或精明细心，多用于人做事认真方面；“精密”指精确细密，多用于仪器的使用方面。“搜集”指到处寻找(事物)并聚集在一起，“收集”多指把现有的东西聚集在一起。
4．依次填入下列各句横线处的成语，最恰当的一项是(　　)
(1)雾霾危及人类的生存环境，对付雾霾不能够____________，民间要有积极行动，政府更应勇于担当。
(2)面对新的发展机遇，我们要转变观念，不能老是____________，守在身边几平方公里的土地上，要走出去，主动迎接种种挑战。
(3)尽管有的人____________，并不以穷为苦，但是穷人家如果遇有债主上门，那就不好受了。
A．乐天知命　安分守己　听天由命
B．乐天知命  听天由命  安分守己
C．听天由命  乐天知命  安分守己
D．听天由命  安分守己  乐天知命
答案　D
解析　乐天知命：相信宿命论的人认为自己的一切都由命运支配，于是安于自己的处境，没有任何忧虑。安分守己：规矩老实，不做超出本分的事。听天由命：任凭事态自然发展变化，不做主观努力，有时也指碰机会或听其自然。
5．下列各句中，没有语病的一项是(　　)
A．一些行业的不良行风如果得不到纠正，社会风气就可能败坏，就难以构建和谐社会、健康社会、幸福社会。
B．在浩如烟海的关于海子的研究和回忆性的文章中，充分说明中国诗歌批评界已经丧失了和真正的海子诗歌世界对话的能力。
C．机动车驾驶人数量的迅猛增长，导致了越来越多的不文明驾驶问题，建立与中国机动车发展相适应的现代交通秩序成为了当务之急。
D．中老年朋友们在冬春季节交替之际应该注重换季调养，按医生专业的养生指导和建议调养锻炼，践行健康的养生之道。
答案　D
解析　A项关联词位置不当，应将“如果”移到句首。B项成分残缺，首句应改为“浩如烟海的关于海子的研究和回忆性的文章”。C项不合逻辑，“机动车驾驶人数量的迅猛增长”与“不文明驾驶问题”不存在因果关系。
