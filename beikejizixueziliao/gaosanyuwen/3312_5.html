2019-2020学年度广东学业水平测试语文复习讲义：(5)


类型三　不合逻辑
不合逻辑是指句子结构完整，搭配合理，在语法方面正确，但不符合概念、判断、推理等形式逻辑或事理逻辑。
【例3】　判断下列句子有无语病，并改正有语病的句子。(有语病×，无语病√)
①浙江省将加大监督力度，改进产品包装和过度包装的做法，推广使用可循环利用物品，限制使用一次性用品。(　　)
②体育运动的目的是身体健康，至于能不能在比赛中得金牌，那是锦上添花的事情。(　　)
③“天地图”的覆盖范围从宏观中国全境到微观具体乡镇、村庄乃至县市。(　　)
④垃圾邮件、木马病毒和低俗内容等三大网络危害已在国内得到有效控制。(　　)
⑤现代化的企业集团往往具备产品的研发、销售、生产及服务能力。(　　)
答案：①×。不合逻辑，“包装”包含“过度包装”，不能并列。
②×。不合逻辑，“能不能”造成两面对一面，把“不能”删去。
③×。不合逻辑，应从大到小，“乡镇、村庄乃至县市”改为“县市、乡镇乃至村庄”。
④√。
⑤不合逻辑，应把“销售”与“生产”调换位置。
说明：(1)表示递进、因果等关系的关联词语所关联的内容是有内在逻辑顺序的，如递进所关联的两个内容，要按照“由易到难，由浅入深，由低到高，由物质到精神”等逻辑顺序来表述。(2)并列词语(短语)形式上并列，但内在的意思有的往往有先后顺序，如第③题。有的无先后顺序，如第④题。
类型四　成分残缺
成分残缺主要有以下几种情况：(1)缺主语；(2)缺谓语；(3)缺宾语；(4)缺状语。
【例4】　判断下列句子有无语病，并改正有语病的句子。(有语病×，无语病√)
①随着航天控制中心科技人员的指令，“嫦娥三号”成功进入预定轨道。(　　)
②因饮酒后驾驶机动车引发的交通事故数量呈快速增长，特别是一些导致多人伤亡的恶性交通事故时有发生。(　　)
③由于史蒂夫·乔布斯的才华和激情，使他在数字音乐、智能手机和平板电脑等领域都作出了革命性的贡献。(　　)
④“天河一号”超级计算机系统具有高自主、高性能、高能效、高安全。(　　)
⑤增进交流，消除隔阂，人类文化一定会展现出共同繁荣、异彩纷呈。(　　)
答案：①√。
②×。宾语残缺，“呈快速增长”后加“的趋势”。
③×。主语残缺，介词“由于”在句首导致主语残缺，删去“由于”。
④×。宾语残缺，应在句子最后加上“的特点”。
⑤×。宾语残缺，在句末加上“的前景”或“的局面”。
说明：一句话的句首如果出现了介词，往往会造成主语的缺失。如第③题。
如果宾语前面的定语较长，那么命题者往往会把宾语中心语去掉从而造成宾语缺失，因为定语太长往往会使我们阅读到最后时丢失了语感，所以在这个地方设置宾语缺失的陷阱是命题者较为青睐的方式之一。成分残缺可用紧缩主干的方式来查找。
