练习16　　语言文字运用+名句名篇默写+论述类文本阅读
一、语言文字运用(15分)
1.(2018无锡期中)在下面一段话的空缺处依次填入的词语,最恰当的一组是(　　)
“　　　　”的社会看上去很有序,但社会体制却是僵化的,人被僵化的体制　　　　,没有自由发挥的空间,一切都是设定好的、　　　　的。这种“有序”的社会是没有活力的社会,当然也是难以向前发展的社会。
A.死气沉沉　束缚　墨守成规
B.死水一潭　束缚　按部就班
C.死水一潭　制约　按部就班
D.死气沉沉　制约　墨守成规
2.(2018兴化一中四模)下列句子中加点熟语使用的修辞手法与其他三项不同的一项是(　　)
A.抗战剧《战地狮吼》讲述的是“狮子黑”赵六从一个行侠仗义的梨园弟子经历父死妻散的悲惨遭遇到投身抗日成长为一代抗日英雄的传奇故事。
B.强盗总打偷菜起,孩子天生是不会有贪念的,也许他们起初觉得拿走别人的东西是一件好玩的事,逐渐就会把“占便宜”“吃白食”等行为视为顺理成章的事了。
C.“向西部小朋友邮寄一本书”活动日前在京启动,孩子们纷纷拿出了自己喜爱的书,寄给远在四川、青海、甘肃等地的小朋友,真是千里送鹅毛——礼轻情意重啊。
D.过去几年来,西藏自治区积极践行绿水青山就是金山银山的理念,将生态文明建设与经济建设和社会发展相结合,建设美丽西藏,促进贫困人口脱贫增收。
3.(2018第二次全国大联考江苏卷)下列各句中,没有语病的一项是　(　　)
A.会议指出,规划建设雄安新区,是以习近平同志为核心的党中央对深化京津冀协同发展作出的又一项重大决策部署,是历史性的一项工程。
B.在此,警方郑重提示,公民个人的行为切莫挑战国家与民族的尊严,在公共场所和公共网络空间的一切行为都不能逾越法律。
C.2月13日,在东印度洋,一艘“商船”遭“海盗”劫持。编队为摸清状况,首先派舰载直升机前去侦破,进而迅速拟制武力营救方案。
D.中国选手武大靖在男子500米短道速滑决赛中,顶住重压,以39秒584的成绩打破世界纪录,拿到中国体育代表团在本届冬奥会上的首金。
4.(2018泰州中学模拟)下列对联,用于某校百年校庆,最恰当的一项是(　　)
A.满腔热血育桃李,一片丹心扶栋梁。
B.画栋拂云联旧垒,玉兰绕砌缀新枝。
C.根深叶茂结硕果,笔健墨浓绘蓝图。
D.福星高照满庭乐,寿诞生辉合家欢。
5.(2018扬州一中模拟)下面文章与栏目名称对应最准确的一项是(　　)
第一组:《天眼计划》《星穹之外》
