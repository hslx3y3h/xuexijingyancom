高三教学学习资料分享（打印版）
仿真模拟(二)
(时间：80分钟　满分：100分)
一、选择题(本大题共16小题，每小题3分，共48分。每小题列出的四个备选项中只有一个是符合题目要求的，不选、多选、错选均不得分)
1．下列加点的字，每对读音都不相同的一项是(　　)
A．雪茄/茄子　古刹/刹车　血统/血汗
B．干劲/劲敌  狙击/沮丧  曲折/歌曲
C．尽管/尽情  捎带/稍微  上校/校对
D．阿谀/婀娜  拙劣/罢黜  着陆/着凉
答案　B
解析　B项依次读jìn/jìnɡ，jū/jǔ，qū/qǔ。A项依次读jiā/qié，chà/shā，xuè。C项依次读jǐn/jìn，shāo，xiào/jiào。D项依次读ē，zhuō/chù，zhuó/zháo。
2．下列各组词语有错别字的一项是(　　)
A．遏制　家喻户晓　英雄气概
B．气侯  微不足道  没精打彩
C．粗犷  不计其数  恪尽职守
D．风韵  物竞天择  一筹莫展
答案　B
解析　侯—候，彩—采。
3．下列句子中加点的词语运用不恰当的一项是(　　)
A．浙江的“茶文化”与“一带一路”接轨，将带动本地区的传统文化走向世界。
B．随着碎片化信息的泛滥、网络语言的流行，简洁、含蓄、内敛的文风逐步式微。
C．道家作为与儒家并驾齐驱的一大流派，在许多方面与儒家形成对立统一的关系。
D．互联网时代，在交通、餐饮等传统经济领域，“分享经济”正如火如荼地发展。
答案　B
解析　式微：借指国家或世族衰落，也泛指事物衰落。一般用作“日渐式微”“渐渐式微”。
4．下列句子没有语病的一项是(　　)
A．为了推进教育改革，强化学生的自我管理意识，这个学校开始施行无人监场。
B．很显然，防备展品免遭盗窃，这是这一次展览会主办者大伤脑筋的一个问题。
C．面对激烈的国际竞争，使我们意识到亟需制定保护知识产权措施的重要性和迫切性。
D．在21世纪，生物基因技术将在人类同各种疾病的斗争中发挥巨大的作用。
答案　D
解析　A项成分残缺，应在“无人监考”后加“的模式”。B项不合逻辑，应去掉“免”。C项缺少主语，应去掉“使”。
5．依次填入下面横线处的句子，最恰当的一项是(　　)
小路上脚印稀罕，不闻人语，它直通故地。谁没有故地？______________，______________。可是谁又会一直心系故地？直到今天我才发现，一个人长大了，走向远方，投入闹市，足迹印上大洋彼岸，他还会固执地指认：____________。
