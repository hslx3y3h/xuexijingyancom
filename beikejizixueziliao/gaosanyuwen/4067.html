2019版浙江省学业水平考试语文仿真模拟全辑（全解析）（10套）模拟(四)高三语文


仿真模拟(四)
(时间：80分钟　满分：100分)
一、选择题(本大题共16小题，每小题3分，共48分。每小题列出的四个备选项中只有一个是符合题目要求的，不选、多选、错选均不得分)
1．下列加点字的读音全都正确的一项是(　　)
A．弥望(mí)　　　　　        瘦削(xiāo)
渺茫(miǎo)          崇山峻岭(chónɡ)
B．点缀(zhuì)          徘徊(huái)
间断(jiān)           惴惴不安(zhuì)
C．踱步(duó)          剥蚀(bō)
敛裾(jū)          含情脉脉(mò)
D．自诩(yǔ)          商埠(bù)
正月(zhēnɡ)          少不更事(ɡēnɡ)
答案　C
解析　A项“削”读xuē。B项“间”读jiàn。D项“诩”读xǔ。
2．下列各组词语没有错别字的一项是 (　　)
A．缅怀　揣摹　耸人听闻
B．寒伧  缥渺  瑕瑜互见
C．频临  盘桓  一蹶不振
D．笼络  瑕疵  负隅顽抗
答案　D
解析　A项摹—摩。B项渺—缈。C项频—濒。
3．下列句子中加点的词语运用不恰当的一项是(　　)
A. 保障房属于深圳住房改革的一种，也是遏制房价、改善住房需求的一种手段。
B．草原的天气变幻莫测，一会儿烈日当空，一会儿大雨倾盆，忽而雨雪交加，忽而狂风怒吼。
C．刚才，我向大家分享了一些我个人读书治学的心得，若有不当之处，敬请各位不吝赐教。
D．实施新政策不到三年时间，咱村就已改头换面，不仅重见青山绿水，而且家家都致富了。
答案　D
解析　A项遏制：制止，控制。使用正确。B项变幻：不规则地改变，多指抽象事物。使用正确。C项不吝赐教：不吝惜自己的意见，希望给予指导。请人指教的客气话。使用正确。D项改头换面：比喻只改形式，不变内容(含贬义)。使用错误。
4．下列句子没有语病的一项是(　　)
A．出租车无论是否使用打车软件，均应遵守运营规则，这样才能维护相关各方的合法权益和合理要求。
B．由于青少年心智尚未成熟，对事物缺乏分辨力，容易被大众媒介中的不良信息诱导，从而产生思想上、行为上的偏差。
C．纪录片《我在故宫修文物》围绕着现代人和古老文物之间的对照互动为核心命题，集中展示了文物修复专家们的工匠精神。
