2020年学业水平考试语文对点备考学案：第一专题(2)


答案　(1)宾语前置句　(2)介宾短语后置句　(3)介宾短语后置句　(4)省略句　(5)省略句　(6)省略句、被动句
一、语言基础及运用
1．下列词语中，每对加点字的读音都不相同的一项是(　　)
A．商贾/贾府　钿头/佃户　云篦/山麓
B．敛容/入殓  怜悯/姓闵  血色/血缘
C．铁骑/轻骑  霓裳/衣裳  红绡/逍遥
D．铮铮/挣脱  嘲哳/浙江  幽咽/咽喉
答案　D
解析　D项zhēnɡ/zhènɡ，zhā/zhè，yè/yān。A项ɡǔ/jiǎ，diàn，bì/lù。B项liǎn/liàn，mǐn，xuè。C项jì/qí，chánɡ/shɑnɡ，xiāo。
2．下列各句中，没有错别字的一项是(　　)
A．醉不成欢残将别，别时茫茫江浸月
B．别有忧愁暗恨生，此时无声胜有声
C．东船西舫悄无言，唯见江心秋月白
D．银瓶炸破水浆迸，铁骑突出刀枪鸣
答案　C
解析　A项残—惨。B项忧—幽。D项炸—乍。
3．对下列句子中加点词的解释，正确的一项是(　　)
A．如听仙乐耳暂明　　　　        暂：暂时
B．却坐促弦弦转急          促：拧紧
C．梦啼妆泪红阑干          阑干：栏杆
D．遂命酒，使快弹数曲          快：赶紧，赶快
答案　B
解析　A项暂：突然。C项阑干：纵横的样子。D项快：畅快，痛快。
4．下列各句中，加点词的意义和用法与现代汉语全都相同的一组是(　　)
①明年秋　②委身为贾人妇　③秋月春风等闲度　④今漂沦憔悴　⑤铁骑突出刀枪鸣　⑥整顿衣裳起敛容　⑦恬然自安　⑧因为长句　⑨凄凄不似向前声
A．①②⑤⑥          B．②③④⑦
C．②③⑧⑨         D．①④⑤⑨
答案　B
解析　①古义：第二年。今义：时间词。今年的下一年。⑤古义：爆发出，突然冲出。今义：鼓出来；超过一般地显露出来；使超过一般。⑥古义：整理。今义：使紊乱的变为整齐，使不健全的健全起来(多指组织、纪律、作风等)。⑧古义：两个词，因而创作。今义：表示接在后面的部分是原因；常跟“所以”搭配使用，表示因果关系。⑨古义：刚才。今义：朝前。
5．下列各句中，加点的诗句引用不正确的一项是(　　)
A．这些老屋虽然有些味道，但“夕阳无限好，只是近黄昏”，新的小区要开发了，不得不推倒它们啊！
B．刚退休不久的老王常对老伴抱怨：“你看我才退下来几天，就‘门前冷落鞍马稀’了，真是人走茶凉啊！”
C．大家出来闯天下，到一起就是缘分，“相逢何必曾相识”，我提议，干杯！
