2020版高考语文（浙江专用）一轮课件：专题三　(17)

答案    D　A项,语序不当,“激活产业,推动智能制造的发展”,应与“解放人力”前后互换。
B项,成分残缺,应在“看作”前加“被”。C项,结构混乱,句式杂糅,可删去“根据”。故
选D。
3.(2019浙南名校联盟,4)下列各句中,没有语病的一项是(3分) (　　)
A.面对更多出行的旅客,机场公安分局始终秉持服务旅客、守护机场的从警理念,不断深化
“平安民航”,为机场的有序运行和建设发展创造平安环境。
B.数据显示,2018年1月—11月期间,加拿大来华人数达到78万人次。发言人陆慷表示,中国是
否安全,加拿大老百姓心中有一杆秤。
C.电动汽车和智能手机、智能家居等市场的迅速扩张,无线电能传输技术将成为一个全新的
机会领域,国家必须适时地支持发展这项技术。
D.温州作为全省网络综合治理体系建设唯一试点,打造首个全国地市级辟谣举报平台——温
州辟谣举报平台,截至目前已累计发布辟谣信息6 000多条。
答案    B　A项,成分残缺,“不断深化……”缺宾语中心语。C项,中途易辙,可在句首加“随
着”。D项,语序不当,“首个”应提到“地市级”前;偷换主语,“发布”的主语应该是“温州
辟谣举报平台”。
4.(2019温州第一次联考,4)下列各句中,没有语病的一项是(3分) (　　)
A.对于2017年业绩大幅下滑的原因,可口可乐解释为2016年公司在业务上有很多变动,尤其在
北美及世界各地将瓶装业务出售给有特许经营权的合作伙伴,以至连总公司都难以控制局面
的发展。
B.前段时间登录央视的电视剧《楼外楼》讲述了楼外楼洪氏兄弟如何秉持“以菜名楼,以文
兴楼”,把楼外楼从一家湖畔饮食小店发展成为名扬天下的餐饮、文化名楼的故事,深受观众
喜爱。
C.众多涉事的疫苗企业,多年游走于监管缝隙,穿越法律、政策防线,这并不是因为其本身有多
大能量,根源还在于后面监管环节的全面失守所致,哪怕有一个环节生效,问题也不至如此严
重。
D.川航3U8633航班在执行重庆至拉萨飞行任务中,驾驶舱右座前挡风玻璃破裂脱落,机长刘传
健等机组成员沉着应对,克服恶劣环境的影响,使飞机成功备降成都双流机场。
答案    D　A.不合逻辑,“北美”和“世界各地”概念有交叉。B.成分残缺,“以菜名楼,以文
兴楼”后面应加“的经营理念”。C.句式杂糅,可去掉“所致”。
5.(2019绍兴3月联考,4)下列各句中,没有语病的一项是(3分) (　　)
A.《南方周末》秉持“在这里,读懂中国”为办报宗旨,以“正义、良知、爱心、责任”为基
本理念,是中国深具公信力与影响力的新闻周报。
