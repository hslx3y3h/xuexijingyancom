2020高考语文二轮复习测试：第1部分 语言文字应

2020高考语文二轮复习测试：第1部分 语言文字应用 专题对练(三)高三语文


　　　　　　　 　　　　　　　　　　　　
1.下列各句中,表达得体的一项是(3分)(　　)
A.昨日收到您的来信,十分高兴。好久不见,甚是想念,我现在工作、身体、家庭都很好,承蒙垂念,本人不胜感激。
B.你来拜会我,我也将拜访你,哪能总是让你三番五次地跑呢?
C.周末是你女儿大婚的日子,我将屈驾前往,你就先忙你的吧,别担心我了。
D.我反复挑选、试穿,售货员不仅不厌其烦,还给我端来茶水,服务真是热情周到,像这样的商店,以后我还会惠顾。
2.下列各句中,表达得体的一项是(3分)(　　)
A.浙江大学建校120周年前夕,学校各部门和各界人士都在积极筹办校庆活动。校学生会建议邀请任职时间较近的5任校长届时组成“男子天团”参加校庆晚会,并且责成校庆筹委会做好落实工作。
B.阿斯塔纳专项世博会开幕之际,国家主席习近平出席了开幕式,代表中国政府向此次专项世博会的开幕致以热烈的祝贺,并预祝大会圆满成功。同时表示,中哈两国关系将会继续巩固和发展下去。
C.马云的处女作出版了,出版社准备为他的处女作召开作品研讨会,马云激动地给著名学者詹教授打电话说:“詹教授,您一定要来呀!我非常期待您的高见,请不要辜负我的一片雅意!”
D.北京电影学院艺考成绩公布了,王俊凯的成绩在所有考生中名列前茅。王俊凯的父母对指导老师说:“张老师,令郎这次能取得优异成绩,多亏您的悉心指导啊!我们感激不尽!”
3.下列各句中,表达得体的一句是(3分)(　　)
A.这次地震的震中距离22日康定6.3级地震震中约十千米远,与康定城区已然十分接近,所幸的是未造成人员伤亡。(广播稿)
B.作为我市唯一的省级贫困县,各级领导必须在号准贫困的“脉”的同时去找准致贫的“根”和开准治贫的“方”,坚决打赢脱贫攻坚战。(会议讲话)
C.松,种子植物的一种,一般为常绿乔木,树脂可提取松香或松节油等;种子可榨油和食用。(工具书)
D.惊悉足球课将成为学校的必修课,我们非常激动,要求学校尽快把足球场建好,让我们有足够大的场所踢球。(写给校长的信)
4.下列各句中,表达得体的一项是(3分)(　　)
A.先生若能拨冗参加我校建校115周年庆祝活动,你将有幸看到精彩演出,并获得纪念品一份。
B.请不要在网络上攻击我们景点服务差了,谁不会偶尔犯个错儿?总盯着有缝的蛋叮的,那是苍蝇。
C.贵刊以提升中学生写作能力为宗旨,在中学生中间还算稍微有点儿影响,我就是贵刊的忠实读者。
D.老张啊,恭贺你乔迁之喜,你这楼房盖得太漂亮了,参观之后,我感觉就剩“羡慕嫉妒恨”了。
