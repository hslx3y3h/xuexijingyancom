2019版浙江省学业水平考试语文对点备考（全WOR(3)


一、语言基础及运用
1．下列词语中，加点字的注音全都正确的一项是(　　)
A．氾南(fàn)　　　佚之狐(yì)　　　　 逢孙(pánɡ)
B．秦伯说(yuè)    失其所与(yù)       贰于楚(èr)
C．共其乏困(ɡònɡ)  缒而出(zhuì)       阙秦(jué)
D．杨孙戍之(shù)  夫晋，何厌之有(fū)  许君焦、瑕(xiá)
答案　A
解析　B项“与”读yǔ。C项“共”读ɡōnɡ。D项“夫”读fú。
2．下列各组句子中，加点词的解释不正确的一组是(　　)
A.
B.
C.
D.
答案　D
解析　说：同“悦”，高兴。
3．下列各句中，加点词的意义与现代汉语相同的一项是(　　)
A．行李之往来，共其乏困
B．若舍郑以为东道主
C．今老矣，无能为也已
D．微夫人之力不及此
答案　C
解析　A项古义：也作“行理”，外交使节。今义：出门所带的包裹、箱子等。B项古义：东方道路上的主人。今义：请客的主人。D项古义：那人，此句中代指秦穆公。今义：对一般人的妻子的尊称。
4．下列各句中，加点词的活用现象与例句相同的一项是(　　)
例句：若亡郑而有益于君
A．因人之力而敝之
B．晋军函陵
C．邻之厚，君之薄也
D．烛之武退秦师
答案　D
解析　例句是动词的使动用法，使……灭亡。A项形容词用作动词，损害。B项名词用作动词，驻军。C项形容词用作动词，变雄厚。D项动词的使动用法，使……撤兵。
5．下列各句中，文言句式与其他三项不同的一项是(　　)
A．以其无礼于晋          B．且贰于楚也
C．夫晋，何厌之有          D．若亡郑而有益于君
答案　C
解析　C项为宾语前置句，其他三项均为介宾短语后置句。
6．下列对文化常识的解说，有误的一项是(　　)
A．是寡人之过也
寡人：秦朝之前的君主自称，被封诸侯王者也可自称“寡人”。
B．敢以烦执事
执事：执行事务的人，对对方的敬称。
C．子亦有不利焉
子：对古代老年男子的尊称。
D．缙绅而能不易其志者
缙绅：也作“搢绅”，指古代的官员，即士大夫。
答案　C
解析　成年男子。
7．在横线上补写出空缺的句子。
(1)晋侯、秦伯围郑，以其无礼于晋，________________。
(2)______________，君知其难也，焉用亡郑以陪邻？
(3)许君焦、瑕，__________________，君之所知也。
(4)______________，不知；______________，不武。
答案　(1)且贰于楚也　(2)越国以鄙远　(3)朝济而夕设版焉　(4)失其所与　以乱易整
