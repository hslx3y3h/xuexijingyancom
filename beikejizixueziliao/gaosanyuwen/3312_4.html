2019-2020学年度广东学业水平测试语文复习讲义：(4)


(13)出现了“前去”“新生”“保管”“没有”“走”“和”等多义词或多义短语，可能是语意不明。
①学校的通知说，让我们20日前去报到。(是“20日之前去”还是“20日这一天去”，语意不明)
②在喧天的锣鼓声中，这所有名的老校终于迎来了自己的新生。(“新生”是指“新同学”还是指“新生命”，语意不明)
③此次选举村民委员会主任，他们谁也没有干涉王尔德的权利。(“没有”兼有副词和动词的性质，造成语意不明)
(14)出现了数量短语，可能是赘余、歧义、不合逻辑。
①他大约五十上下。(“大约”与“上下”重复)
②几个学校领导出席了会议。(“几个学校领导”有歧义)
附：常见的六种病句类型
类型一　结构混乱
结构混乱有以下几种表现形式：(1)句式杂糅：①表被动；②表原因；③表目的；④表条件；⑤表意图；⑥表对象；⑦表构成；⑧表范围；⑨表程度；⑩表数量。(2)双主语。(3)层次不清。
【例1】　判断下列句子有无语病，并改正有语病的句子。(有语病×，无语病√)
①广大政协委员围绕以“五水共治”等问题为议题，积极献言献策。(　　)
②广州恒大足球队的大手笔投入，夺得了去年的亚冠冠军和中超冠军。(　　)
答案：①×。(句式杂糅，“围绕”“以……为议题”删去一个。)
②×。(改为“大手笔投入的广州恒大足球队”。)
说明：偷换主语导致结构混乱，紧缩每个分句和句子的主干可以判断结构混乱的语病。
类型二　重复赘余
重复赘余有以下几种表现形式：(1)语义重复；(2)附加成分(定语、状语、补语)赘余；(3)主语、谓语、宾语中同义成分赘余；(4)滥用介词；(5)滥用关联词语。
【例2】　判断下列句子有无语病，并改正有语病的句子。(有语病×，无语病√)
①去年物价总体保持平稳，数据表明12月份CPI在不到3%左右运行。(　　)
②“国庆”期间，杭州湾跨海大桥的车流量增大，平均每天超过4万多辆。(　　)
③电视连续剧《辛亥革命》展现了一幅宏大厚重的历史画卷，成为纪念辛亥革命100周年活动的一份厚礼。(　　)
④解决老百姓看病贵问题的关键，最主要的办法是制定国家基本药物目录。(　　)
答案：①×。语意重复(不合逻辑)，“不到3%左右”改为“不到3%”或“3%左右”。
②×。重复赘余，“超过”与“多”重复，删去“多”。
③√。
④×。重复赘余，“关键”与“最主要的方法”重复。
说明：语意重复往往在并列包含或互相交叉的词语间设误，或者在递进关系的两者之间设置误区。所以要重点关注句子中表递进关系的关联词；关注形式上并列的短语，这样的短语往往在意义上有层递的关系。
