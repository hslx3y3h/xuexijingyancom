仿真模拟(六)
(时间：80分钟　满分：100分)
一、选择题(本大题共16小题，每小题3分，共48分。每小题列出的四个备选项中只有一个是符合题目要求的，不选、多选、错选均不得分)
1．下列加点字的读音全都正确的一项是(　　)
A．敕造(chì)　　　　　　        呜咽(yàn)
促膝谈心(qī)          陈抟老祖(tuán)
B．瞋视(chěn)          兑现(duì)
自怨自艾(yì)          长歌当哭(dànɡ)
C．模样(mú)          熨帖(yù)
熠熠闪光(yì)          转弯抺角(mò)
D．叱咤(zhà)          咯血(kēi)
孝悌之义(tì)          曲水流觞(shānɡ)
答案　C
解析　A项“咽”读yè，“膝”读xī。B项“瞋”读chēn。D项“咯”读kǎ。
2．下列各组词语没有错别字的一项是(　　)
A．寒喧　　　　始料不及　　　　万籁有声
B．徇私　　　　梦寐以求　　　　前倨后恭
C．和霭　　　　舍本逐末　　　　喜笑怒骂
D．萦怀　　　　义愤填膺　　　　隐诲曲折
答案　B
解析　A项喧—暄。C项霭—蔼，喜—嬉。D项诲—晦。
3．下列句子中加点的词语运用恰当的一项是(　　)
A．一个人的成长，就是磨砺的过程，被小病小灾磨，被贫穷困苦磨，被挫折坎坷磨，或者从小到大，锦衣玉食，万事顺遂，亦不免被光阴磨。
B．青春期的少年正处于心理断乳期，控制能力差，相对于成年人更需要规则的约束，校园暴力的频发正影射出校园规则的失范。
C．真理从来不佶屈聱牙，而是返璞归真，我们穷尽一生想要抵达的境界，其实只是花开的片段、叶落的瞬间。凡尘碌碌，要学会如何与世界相处。
D．幸福不喜欢喧嚣浮华，它常常在平淡中降临。贫困中相濡以沫的一块糕饼，患难中心心相印的一个眼神，都是千金难买的幸福。
答案　D
解析　A项或者：或许；用在叙述句里，表示选择关系；表示等同关系。用在此处不合语境，应用“纵使”。B项影射：借甲指乙；暗指(某人某事)。用在此处不合语境，应用“折射”。C项佶屈聱牙：(文章)读起来不顺口。用在此处对象错误。D项相濡以沫：比喻同处困境，相互救助。用在此处恰当。
4．下列句子没有语病的一项是(　　)
A．演艺界的明星风光无限，但背后的竞争激烈而残酷，他们没有属于自己的私生活，许多人靠的是利用安眠药度日，为自己的健康埋下祸患。
