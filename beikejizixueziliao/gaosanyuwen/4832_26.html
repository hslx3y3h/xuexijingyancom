2020版高考语文浙江专用一轮课件：专题二词语的(26)


(1)文段中加点的词,运用不正确的一项是(3分) (　　)
A.辑录　　B.大而无当
C.温情　　D.需
(2)文段中画线的甲、乙、丙三句,标点有误的一项是(2分) (　　)
A.甲　　　　B.乙　　　　C.丙
答案　(1)C　(2)B   
解析    (1)“温情”为名词,此处应用“温暖”等形容词。
(2)“人性”后面的逗号应改为分号,以区分并列复句之间的层次。
4.(2019杭州第一学期质量检测,2—3)阅读下面的文字,回答问题。(5分)
正当防卫与紧急避险交叉行为所引起的问题,在众所周知的玛丽与乔迪连体双胞胎案中显露
出来。[甲]玛丽的心脏停止了跳动,只能依靠乔迪的心脏供血;乔迪的心脏因此而不堪重负,需
要做手术才能挽救乔迪的生命。[乙]现在的问题是,使连体婴儿中较强壮的那个幸存下来的
机会最大化,其附带结果是较为弱小的那个就得死亡,那么通过手术将这对连体双胞胎姐妹俩
分开是否合法呢?从表面上看,分体手术是非法的,构成谋杀罪,因为虽然手术是为了挽救乔迪
的生命,但肯定是对玛丽的杀害,并且,从玛丽自身利益的观点来看,不存在任何使该手术值得
去做的补偿措施并因此明确地证明医疗干涉具有正当性。尽管如此,两害相权取其轻,很多人
的道德直觉还是赞成分体手术。[丙]但是,这种简单功利计算的方法忽略了一个基本事实:至
少忽略了医生不能在未经病人同意的情况下仅仅为了病人的利益而采取重大的医疗行为。
(1)文段中的加点词语,运用不正确的一项是(3分) (　　)
A.众所周知　　   B.并且           C.干涉　   　D.两害相权取其轻
(2)文段中画线的甲、乙、丙句,标点有误的一项是(2分)      (　　)
A.甲　　          　B.乙　　           C.丙
答案　(1)C　(2)C
解析    (1)众所周知:大家全都知道。此处使用正确。并且:表示更进一层。使用正确。干涉:
一个意思是“过问或制止,多指不应该管硬管”,另一个意思是“关涉,关联”。这两个意思用
在此文中都不合适。两害相权取其轻:两种选择都有危害,但必须选择一种,那么就选择危害轻
一点的。使用正确。
(2)冒号应改为逗号。冒号表示后面的话是解释,这前后两句话是在说同一件事情的两个方面,
后一句在前一句的基础上有点让步的意味。
5.(2019温州第二次联考,2—3)阅读下面的文字,回答问题。(5分)
社评写作强调理性,文学写作强调感性。[甲]理性和感性之间,在很多人那里,是水星与火星的
距离;但在金庸那里,则水乳交融,切换自如,令人叹为观止。
