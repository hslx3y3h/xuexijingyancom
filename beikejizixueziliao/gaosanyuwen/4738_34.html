3.文中画横线的句子有语病,下列修改最恰当的一项是(3分) (　　)
A.激发中华优秀传统文化的内在以及提升国家文化软实力和中华文化影响力,从而彰显文化自信。
B.激发中华优秀传统文化的内在生命力,提升国家文化软实力和中华文化影响力,从而彰显文化自信
C.激发中华优秀传统文化的内在生命力以及国家文化软实力和中华文化影响力,从而彰显文
化自信。
D.激发中华优秀传统文化的内在,提升国家文化软实力和中华文化影响力,从而彰显文化自信。
三、
1.答案    C　根据前后文的逻辑可知,括号内所填内容应以“传承的……”开头,这样才能使
上下句连贯,故可排除A、B两项。D项“不只是……而是……”不搭配,且与原文要表达的意
思不符。所以选C。
2.答案    C　潜移默化:人的思想或性格受其他方面的感染而不知不觉地起了变化。耳濡目
染:形容听得多见得多了之后,无形之中受到影响。与时俱进:随着时代的发展而不断发展、前
进。强调的是顺应时代的发展和潮流。别开生面:另外开展新的局面或创造新的形式。强调
的不是“顺应”,而是“创新和不同”。喜闻乐见:喜欢听,乐意看,形容很受欢迎,侧重于对所
“闻”“见”的事物的态度。“津津乐道”形容对某件事情很感兴趣,乐于谈论,强调的是说
个不停。古为今用:批判地继承古代优秀的东西,使之为今天服务,侧重于“运用”。借古论
今:假托古代的事物讨论现实,侧重于“论说”。
3.答案    B　“激发中华优秀传统文化的内在以及国家文化软实力和中华文化影响力,从而彰
显文化自信。”这个句子的主干是“激发内在以及软实力和影响力”,明显“激发”与“内
在”不搭配,缺宾语中心语;“激发”与“软实力和影响力”也不搭配,应是“提升”“软实力
和影响力”。所以选B。
[创新点]    素材创新。选取体现中华传统文化特色的材料,考查学生语言运用、成语选用、
病句修改等能力。
