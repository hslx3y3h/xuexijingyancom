2020版高考语文全程训练计划-小题狂练-天天练9高(3)


D．国家电力调度控制中心昨天发布的数据显示，国家电网公司区域内多个电网用电负荷超过夏季用电高峰，突破历史极值。
答案：D
解析：A项，语序不当，“在全省范围内”“自12月7日起”调换位置，状语顺序应该先时间，后范围。B项，成分残缺，“含有有毒化学物质”前应加上“蓝雪中”。C项，结构混乱，前面的主语是“公司规模”，没有说完，后面又出现“孙晓栋”这一新的主语，中途易辙。
能力提升
7．下列各句中，没有语病的一项是(　　)
A．随着国家经济实力的增强，学术投入的增加，研究的物质条件大为改观，博士生待遇明显提高，“艰苦”二字已非学术研究的同义语。
B．这次从重庆前往南极考察的共10人，他们都是重庆人，平均年龄近50岁左右，组织这次考察活动的是位于渝北的一家旅游机构。
C．如何真正把农民工留在自己家门口就业，用工企业不仅要着实提供良好的待遇，而且政府也要搭建好平台。
D．昆曲唱腔清丽婉转，恰似一位婀娜多姿、风情万种的美人，几乎可以和西方同一时期的戏剧经典著作相媲美。
答案：A
解析：B项，不合逻辑，“近”与“左右”矛盾，保留其一。C项，关联词位置不当，应将“不仅”放到“用工企业”前面。D项，偷换主语，可在“几乎”前加“其作品”。
8．下列各句中，没有语病的一项是(　　)
A．日前，我国最大规模跨海输电高塔群即将正式投入运行，高塔群起自岱山岛蓬莱220千伏变电站，终至定海侧开接点，平均塔高192.8米。
B．每个孩子在其成长过程中都会发生一些错误，作为家长，我们要有耐心，懂方法，让孩子认识到这些错误，并加以改正。
C．人工合成的酿酒酵母染色体，能够为癫痫、癌症、智力发育迟缓和衰老等人类面临的医学难题提供研究与治疗模型。
D．微电影是在新媒体平台上播放，具有完整策划和系统制作过程、完整故事情节，可以单独成片，也可以系列成剧。
答案：C
解析：A项，不合逻辑，“日前”是“几天前”的意思，与“即将”前后矛盾。B项，搭配不当，“发生”和“错误”不搭配，可将“发生”改为“犯”。D项，成分残缺，“微电影”缺少宾语中心语，可在“故事情节”后加上“的短片”。
9．下列各句中，没有语病的一项是(　　)
A．我国将全面建立省、市、县、乡四级河长体系，构建责任明确、协调有序、监管严格、防护有力的河湖管理保护。
B．《中国酒文献集成》对历代酒文献进行了全面系统的整理，这不仅对开发酒文化资源、培养健康的饮酒习惯有积极意义，而且便于学界研究使用。
C．改革开放以来，我国保健食品工业迅猛发展，许多食品在世界上享有很高声誉，特别是一些药膳食品的调理滋补，很受青睐。
