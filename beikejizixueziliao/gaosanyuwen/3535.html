2020高考语文二轮复习测试：第2部分 古诗文阅读 保温常练(七)高三语文


一、语言基础知识(9分)
阅读下面的文字,完成1~3题。
　　在我们的日常语言中,真、善、美这三个词常常是　　　　的。仔细寻思后不难发现,自然的根本性质是真,人生的根本意义表现为善,艺术的根本价值在于美。如此看来,真、善、美之间似乎并没有直接的联系,但是在审美的意义上,三者却　　　　。自然于　　　　中感发和激荡着人的情性与思绪,也安顿和抚慰着人的心灵;人则通过审美的思维方式达到情景交融、物我统一的和谐境界。可以说,一片自然山水就是一个人栖居诗意的寓所心灵与家园精神。更为重要的是,每个人的人生都是一道独特的风景,具有审美价值,而且整个审美活动的出发点和落脚点就在于观照人的生存境况,并努力成就自由完美的人生,使人生得以丰富地展开。那么,审美人生的具体内容是什么呢?在我看来,审美意义上的人生,(　　　　);另一方面,主体还以人的社会特质为基础,又不滞于人的社会特质,从心灵中获得精神自由的境界。这种体道的境界与精神自由的境界在审美思维方式上的　　　　,即审美的人生境界,这是对现实人生的积极拓展、提升与超越。
1.依次填入文中横线上的成语,全部恰当的一项是(3分)        (　　)
A.相提并论　　休戚相关　　耳濡目染　　相与为一
B.相提并论        息息相关        潜移默化        相与为一
C.同日而语        息息相关        耳濡目染        融会贯通
D.同日而语        休戚相关        潜移默化        融会贯通
2.文中画横线的部分有语病,下列修改最恰当的一项是(3分)        (　　)
A.一片自然山水就是一个人诗意栖居的心灵寓所与精神家园。
B.一片自然山水就是一个人诗意栖居的寓所心灵与家园精神。
C.一片自然山水就是一个人栖居诗意的心灵寓所与家园精神。
D.一片自然山水就是一个人诗意栖居的寓所心灵与精神家园。
3.下列在文中括号内补写的语句,最恰当的一项是(3分)        (　　)
A.一方面不滞于感性生命,以自然的感性生命为基础,由切身感悟和内省体验而达到与宇宙精神合一的体道境界
B.一方面是指主体以自然的感性生命为基础,又不滞于感性生命,由切身感悟和内省体验而达到与宇宙精神合一的体道境界
