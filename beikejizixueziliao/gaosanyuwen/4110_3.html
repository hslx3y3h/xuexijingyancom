D．①与②意思不同，③与④意思也不同
答案　C
解析　①句中的“三”是虚指，“三岁”指多年。②句中的“三”是实指，“二三”意为三心二意。③④两句中的“靡”的意思都是“无，不”。
4．下列各句中，加点词的古今意义相同的一项是(　　)
A．送子涉淇，至于顿丘
B．淇水汤汤，渐车帷裳
C．不见复关，泣涕涟涟
D．将子无怒，秋以为期
答案　B
解析　A项古义：到，到达。今义：表示达到某种程度或表示另提一事。C项古义：眼泪。今义：鼻涕。D项古义：把……作为。今义：认为。
5．下列各句中，没有词类活用现象的一项是(　　)
A．士也罔极，二三其德
B．桑之落矣，其黄而陨
C．言既遂矣，至于暴矣
D．自我徂尔，三岁食贫
答案　C
解析　A项“二三”，数词的使动用法。B项“黄”，形容词用作动词。D项“贫”，形容词用作名词。
6．在横线上补写出空缺的句子。
(1)氓之蚩蚩，____________。匪来贸丝，____________。
(2)淇水汤汤，____________。女也不爽，____________。
答案　(1)抱布贸丝　来即我谋　(2)渐车帷裳　士贰其行
二、阅读理解训练
阅读课文，然后回答问题。
氓
《诗经》
氓之蚩蚩，抱布贸丝。匪来贸丝，来即我谋。送子涉淇，至于顿丘。匪我愆期，子无良媒。将子无怒，秋以为期。
乘彼垝垣，以望复关。不见复关，泣涕涟涟。既见复关，载笑载言。尔卜尔筮，体无咎言。以尔车来，以我贿迁。
桑之未落，其叶沃若。于嗟鸠兮，无食桑葚。于嗟女兮，无与士耽。士之耽兮，犹可说也。女之耽兮，不可说也。
桑之落矣，其黄而陨。自我徂尔，三岁食贫。淇水汤汤，渐车帷裳。女也不爽，士贰其行。士也罔极，二三其德。
三岁为妇，靡室劳矣。夙兴夜寐，靡有朝矣。言既遂矣，至于暴矣。兄弟不知，咥其笑矣。静言思之，躬自悼矣。
及尔偕老，老使我怨。淇则有岸，隰则有泮。总角之宴，言笑晏晏。信誓旦旦，不思其反。反是不思，亦已焉哉！
7．下列各组句子中，加点词的意义和用法相同的一组是(　　)
A.          B.
C.          D.
答案　A
解析　A项用于主谓之间，没有实在意义。B项连词，表目的/介词，相当于“用”。C项代词，代第三人称/助词，无义；也可作为代词，代第一人称。D项代词/助词，的。
8．对下列两组句子中加点词的意义和用法的判断，正确的一项是(　　)

A．两个“尔”相同，两个“彼”相同
B．两个“尔”不同，两个“彼”相同
C．两个“尔”相同，两个“彼”不同
